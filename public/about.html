Delved into back-end technologies like Node.js and Express. Learned to build dynamic web applications and interact with databases.

Explored popular front-end frameworks like React and back-end frameworks like Django. Built complex applications with ease.

Present: Full-Stack Expertise

Currently, I possess full-stack development expertise, allowing me to create end-to-end solutions. I am passionate about staying updated with the latest web technologies.
